About Back-End

Web APIs... Modern applications have have a very defined separation between Front-End (browser application) and Back-End (server programming). Web APIs are the gateway to server programming. Frame(s)work has developed a number of tools and techniques that simplify their development.

Frame(s)work implements a design for simplifying the complications of many controllers with even more methods and even more method parameters that require complicated documentation. Frame(s)work has a Single Controller with a Single Method and a Single Parameter. The key to this design is its use of 'Dependency Injection'.
